This repository hosts my [Helm](https://helm.sh) charts.

# Usage

```
$ helm repo add atrakic https://atrakic.github.io/helm-charts/
$ helm repo update
$ helm search repo atrakic
$ helm install my-release atrakic/<chart>
```
